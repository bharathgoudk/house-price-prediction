# 🏠 House Price Prediction Web App

This project is a machine learning-based web application that predicts house prices based on features like square footage, number of bedrooms, bathrooms, year built, lot size, garage size, and neighborhood quality.

## 🚀 Features
- Predict house prices using ML model
- Simple and user-friendly interface
- Real-time prediction using Flask

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Flask
- HTML

## ▶️ Run Locally
pip install -r requirements.txt  
python app.py  

Then open:  
http://127.0.0.1:5000/
